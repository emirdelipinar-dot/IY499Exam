Programme Description
This Python-based application is designed for Grouped Data Analysis within an academic or professional context. It allows users to input continuous numerical data (student grades ranging from 0 to 100), which is then stored securely in a CSV format.

The application specialises in converting raw, unstructured data into meaningful statistical insights. By allowing the user to specify a custom class width, the programme performs data binning to generate a frequency distribution. It calculates key measures of central tendency (Mean, Median, and Mode) and dispersion (Variance and Standard Deviation). Furthermore, it identifies the Modal Class and calculates class Midpoints.

To enhance data interpretation, the application provides a robust visualisation tool using Matplotlib to generate a histogram, alongside a tabular frequency report which is exported to a separate CSV file for further study.

The following packages must be installed to run the application:

pandas: For data manipulation and CSV handling.

numpy: For numerical range and bin generation.

matplotlib: For high-quality data visualisation.

statistics: (Built-in) For mathematical computations.

You can type this to your terminal to install the libraries:

pip install pandas numpy matplotlib
